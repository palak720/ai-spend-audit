# Metrics (200-500 words)

## North Star metric
Qualified Savings Leads per Week (QSL/W)

## 3 input metrics
- Audit completion rate
- Lead capture rate after result view
- Share-link generation rate

## First instrumentation
- `audit_started`
- `audit_completed`
- `result_viewed`
- `lead_submitted`
- `share_link_created`
- `consultation_clicked`

## Pivot trigger
If >500 completed audits produce <3% qualified leads for 2 consecutive weeks, revisit recommendation quality and target audience.
